# Liquea Roadmap

## 0.2 - Daily Browsing Foundations

- Session and window restoration
- Private browsing windows
- Per-site permission controls
- Find-in-page overlay
- Download pause, resume, cancel, and retry
- Better certificate and connection details
- Tab audio indicators, mute, pinning, and duplication

## 0.3 - Organization

- Tab groups and workspaces
- Bookmark folders and import/export
- Searchable history
- Multiple browser profiles
- Custom new-tab images
- Reader mode and page translation

## 0.4 - Privacy and Safety

- Tracker and ad blocking using audited filter-list infrastructure
- Per-site cookie and storage dashboard
- HTTPS-only mode
- Configurable telemetry policy with telemetry off by default
- Safer download reputation checks

## 0.5 - Ecosystem

- Password manager integration
- End-to-end encrypted optional sync
- Carefully scoped extension compatibility research
- Signed auto-updates
- macOS notarization and Windows code signing
- Reproducible release pipeline and SBOM generation

## 1.0 - Stable

- Accessibility audit
- Long-running memory and crash testing
- Cross-platform interaction and packaging test matrix
- Documented LiquidOS port after its native APIs are available
- Stable migration policy for user data
